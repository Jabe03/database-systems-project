Database Queries Showcase

1. Tutoring Sessions

Description: This Query simply retrieves a list of all tutoring sessions.


SELECT ts.SessionID, ts.SessionDate,
       s.FirstName AS StudentFirstName, s.LastName AS StudentLastName,
       t.FirstName AS TutorFirstName, t.LastName AS TutorLastName
FROM TutorSession ts
JOIN Students s ON ts.StudentID = s.StudentID
JOIN Tutors t ON ts.TutorID = t.TutorID;

2. Tutor Average Ratings

Description:This query uses an aggregation to show the average rating for each of the tutors.


SELECT T.TutorID, T.FirstName, T.LastName,
       AVG(R.Rating) AS AvgRating
FROM Tutors T
JOIN Review R ON T.TutorID = R.TutorID
GROUP BY T.TutorID, T.FirstName, T.LastName;

3. Tutor Courses

Description: This query uses a join to create a list of all tutors as well as the courses that they teach.


SELECT T.FirstName, T.LastName, C.CourseName
FROM Tutors T
JOIN Teaches Te ON T.TutorID = Te.TutorID
JOIN Courses C ON Te.CourseID = C.CourseID;

4. Students enrolled in a class

Description: Uses a subquery to view a list of all students that are actively enrolled in any course.


SELECT FirstName, LastName
FROM Students
WHERE StudentID IN (
    SELECT StudentID FROM Enrollment
);

5. Tutor Performance

Description: We created a view that lists all tutors and their average ratings. This is a simple query that shows us a list of all tutors whose average rating is greater than or equal to 4. 


SELECT * FROM tutorPerformance
WHERE AverageRating >= 4;
